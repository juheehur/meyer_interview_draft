## 이미지 최적화 가이드:

1. **파일 형식**: PNG 또는 WebP 권장
2. **파일 크기**: 가능한 한 작게 (300KB 이하 권장)
3. **해상도**: 레티나 디스플레이를 위해 2x 크기로 준비
4. **압축**: TinyPNG 또는 similar tools 사용 권장

## 사용법:

이미지 파일들을 이 폴더에 추가한 후, 웹사이트에서 다음과 같이 참조됩니다:

```jsx
<Image
  src="/images/hero-team-cooking.png"
  alt="Team Cooking"
  width={1920}
  height={1080}
/>
```
